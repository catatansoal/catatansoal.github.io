Prove ⌊5x⌋ + ⌊5y⌋ ≥ ⌊3x+y⌋ + ⌊x+3y⌋ For Non Negative X And Y

Hey guys! Today, we're diving deep into a fascinating inequality involving floor functions. Specifically, we're going to prove that ⌊5x⌋ + ⌊5y⌋ ≥ ⌊3x+y⌋ + ⌊x+3y⌋ for all non-negative real numbers x and y. This inequality pops up quite often in contest math, and understanding how to tackle it can be a real game-changer. We'll break down the problem step by step, explore different approaches, and make sure you grasp the core concepts. So, buckle up, and let's get started!

Understanding the Problem

Before we jump into the proof, let's make sure we're all on the same page. The floor function, denoted by ⌊z⌋, gives you the greatest integer less than or equal to z. For example, ⌊3.14⌋ = 3, ⌊5⌋ = 5, and ⌊-2.7⌋ = -3. Understanding this function is crucial for tackling this problem. The inequality itself states that the sum of the floor of 5 times x and 5 times y is always greater than or equal to the sum of the floors of (3x + y) and (x + 3y), provided that x and y are non-negative. This might seem a bit abstract at first, but we'll see how it plays out with some concrete examples later on.

Why is this important? Inequalities involving floor functions often appear in mathematical competitions and problem-solving scenarios. They require a solid understanding of number theory and real analysis, making them excellent tools for sharpening your mathematical skills. By tackling this inequality, we'll not only prove a specific result but also develop techniques that can be applied to a broader range of problems. We'll explore different strategies, highlighting the nuances and potential pitfalls along the way. So, let's break it down and make it super clear!

Initial Thoughts and Strategies

When faced with an inequality involving floor functions, a common strategy is to work with the fractional part. Remember, any real number z can be written as the sum of its integer part (the floor) and its fractional part. Let's denote the fractional part of z as {z} = z - ⌊z⌋. This means 0 ≤ {z} < 1. This decomposition allows us to separate the integer and fractional components, which can be incredibly useful in simplifying the inequality. Another approach involves leveraging the properties of the floor function, such as ⌊a + b⌋ ≥ ⌊a⌋ + ⌊b⌋, which we'll use extensively in our proof. We might also consider using casework or specific examples to gain intuition about the inequality. Before diving into a rigorous proof, it's often beneficial to play around with some values of x and y to see if the inequality holds true. This can give us a sense of the problem and guide our proof strategy.

Proof using Fractional Parts

Let's dive into the heart of the matter and construct a formal proof. As we discussed, the fractional part approach is a powerful tool for handling floor functions. To begin, let's express x and y in terms of their integer and fractional parts:

  • x = ⌊x⌋ + {x}
  • y = ⌊y⌋ + {y}

where ⌊x⌋ and ⌊y⌋ are the integer parts of x and y, respectively, and {x} and {y} are their fractional parts, with 0 ≤ {x} < 1 and 0 ≤ {y} < 1.

Now, let's rewrite the inequality in terms of these fractional parts. We want to prove:

⌊5x⌋ + ⌊5y⌋ ≥ ⌊3x + y⌋ + ⌊x + 3y⌋

Substitute x and y with their fractional parts:

⌊5(⌊x⌋ + {x})⌋ + ⌊5(⌊y⌋ + {y})⌋ ≥ ⌊3(⌊x⌋ + {x}) + (⌊y⌋ + {y})⌋ + ⌊(⌊x⌋ + {x}) + 3(⌊y⌋ + {y})⌋

Using the property ⌊a + b⌋ ≥ ⌊a⌋ + ⌊b⌋, we can further break down the inequality. Notice that ⌊5⌊x⌋ + 5{x}⌋ = 5⌊x⌋ + ⌊5{x}⌋ because 5⌊x⌋ is an integer. Similarly, ⌊5⌊y⌋ + 5{y}⌋ = 5⌊y⌋ + ⌊5{y}⌋. So the left-hand side (LHS) becomes:

LHS = 5⌊x⌋ + ⌊5{x}⌋ + 5⌊y⌋ + ⌊5{y}⌋

For the right-hand side (RHS), we have:

RHS = ⌊3⌊x⌋ + 3{x} + ⌊y⌋ + {y}⌋ + ⌊⌊x⌋ + {x} + 3⌊y⌋ + 3{y}⌋

Using the property ⌊a + b⌋ ≥ ⌊a⌋ + ⌊b⌋ again, we get:

RHS ≥ ⌊3⌊x⌋ + ⌊y⌋ + 3{x} + {y}⌋ + ⌊⌊x⌋ + 3⌊y⌋ + {x} + 3{y}⌋

RHS ≥ 3⌊x⌋ + ⌊y⌋ + ⌊3{x} + {y}⌋ + ⌊x⌋ + 3⌊y⌋ + ⌊{x} + 3{y}⌋

RHS ≥ 4⌊x⌋ + 4⌊y⌋ + ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

Now, we need to show that:

5⌊x⌋ + ⌊5{x}⌋ + 5⌊y⌋ + ⌊5{y}⌋ ≥ 4⌊x⌋ + 4⌊y⌋ + ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

This simplifies to:

⌊x⌋ + ⌊y⌋ + ⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

This is where the clever part comes in. We've reduced the problem to an inequality involving only the fractional parts. Since 0 ≤ {x} < 1 and 0 ≤ {y} < 1, we can focus on proving:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

This inequality is true because ⌊5{x}⌋ ≥ ⌊3{x} + {y}⌋ and ⌊5{y}⌋ ≥ ⌊{x} + 3{y}⌋. This isn't immediately obvious, so let's break it down further in the next section.

Proving the Fractional Part Inequality

The crux of our proof now rests on demonstrating that:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

where 0 ≤ {x} < 1 and 0 ≤ {y} < 1. To tackle this, we can consider different cases based on the values of {x} and {y}. A common strategy is to divide the unit square (0 ≤ {x} < 1, 0 ≤ {y} < 1) into regions where the floor functions take on constant values. This allows us to analyze the inequality in a piecewise manner.

Casework Approach

Let's divide the possible values of {x} and {y} into smaller intervals. Since we have terms like ⌊5{x}⌋ and ⌊5{y}⌋, it's helpful to consider intervals of length 1/5. We'll analyze the inequality based on the ranges of 5{x} and 5{y}.

Case 1: 0 ≤ {x} < 1/5 and 0 ≤ {y} < 1/5

In this case, ⌊5{x}⌋ = 0 and ⌊5{y}⌋ = 0. Also, 3{x} + {y} < 3(1/5) + 1/5 = 4/5 < 1, so ⌊3{x} + {y}⌋ = 0. Similarly, {x} + 3{y} < 1/5 + 3(1/5) = 4/5 < 1, so ⌊{x} + 3{y}⌋ = 0. Thus, the inequality holds: 0 + 0 ≥ 0 + 0.

Case 2: 1/5 ≤ {x} < 2/5 and 0 ≤ {y} < 1/5

Here, ⌊5{x}⌋ = 1 and ⌊5{y}⌋ = 0. We have ⌊3{x} + {y}⌋. Since 3{x} + {y} < 3(2/5) + 1/5 = 7/5, ⌊3{x} + {y}⌋ can be either 0 or 1. Also, ⌊{x} + 3{y}⌋ = 0 since {x} + 3{y} < 2/5 + 3(1/5) = 1. Now, let's consider the worst-case scenario where ⌊3{x} + {y}⌋ = 1. Even in this case, 1 + 0 ≥ 1 + 0, so the inequality holds.

Case 3: 0 ≤ {x} < 1/5 and 1/5 ≤ {y} < 2/5

This case is symmetric to Case 2, and the inequality also holds.

Case 4: 1/5 ≤ {x} < 2/5 and 1/5 ≤ {y} < 2/5

In this case, ⌊5{x}⌋ = 1 and ⌊5{y}⌋ = 1. We have 3{x} + {y} < 3(2/5) + 2/5 = 8/5, so ⌊3{x} + {y}⌋ can be 0 or 1. Similarly, {x} + 3{y} < 2/5 + 3(2/5) = 8/5, so ⌊{x} + 3{y}⌋ can also be 0 or 1. The worst-case scenario is when both ⌊3{x} + {y}⌋ = 1 and ⌊{x} + 3{y}⌋ = 1. But even then, 1 + 1 ≥ 1 + 1, so the inequality holds.

We can continue this casework for all possible ranges of {x} and {y}. However, this approach can become quite tedious. There's a more elegant way to tackle this using the properties of the floor function directly.

A More Elegant Approach

Instead of a brute-force casework approach, we can leverage the following property of the floor function:

⌊a⌋ + ⌊b⌋ ≤ ⌊a + b⌋ ≤ ⌊a⌋ + ⌊b⌋ + 1

Let's apply this property strategically. We want to show:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

Notice that:

⌊5{x}⌋ = ⌊3{x} + 2{x}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋

⌊5{y}⌋ = ⌊3{y} + 2{y}⌋ ≥ ⌊3{y}⌋ + ⌊2{y}⌋

So, we have:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋ + ⌊3{y}⌋ + ⌊2{y}⌋

Now, we need to show that:

⌊3{x}⌋ + ⌊2{x}⌋ + ⌊3{y}⌋ + ⌊2{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

Let's analyze ⌊3{x}⌋ + ⌊2{x}⌋. Since 0 ≤ {x} < 1:

  • If 0 ≤ {x} < 1/3, then ⌊3{x}⌋ = 0 and ⌊2{x}⌋ = 0.
  • If 1/3 ≤ {x} < 1/2, then ⌊3{x}⌋ = 1 and ⌊2{x}⌋ = 0.
  • If 1/2 ≤ {x} < 1, then ⌊3{x}⌋ ≥ 1 and ⌊2{x}⌋ ≥ 1.

Similarly for ⌊3{y}⌋ + ⌊2{y}⌋. This gives us a good starting point for a more focused casework, but we can refine it further.

Final Step: A Direct Comparison

Let's go back to our fractional part inequality:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋

We can see that 5{x} = 3{x} + {x} + {x} and 5{y} = 3{y} + {y} + {y}. This suggests we might be able to compare terms directly. We can use the property ⌊a + b⌋ ≤ ⌊a⌋ + ⌊b⌋. Applying this property, we have:

⌊3{x} + {y}⌋ ≤ ⌊3{x}⌋ + ⌊{y}⌋

⌊{x} + 3{y}⌋ ≤ ⌊{x}⌋ + ⌊3{y}⌋

Adding these inequalities:

⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋ ≤ ⌊3{x}⌋ + ⌊{y}⌋ + ⌊{x}⌋ + ⌊3{y}⌋

Now, we want to show that:

⌊5{x}⌋ + ⌊5{y}⌋ ≥ ⌊3{x}⌋ + ⌊{y}⌋ + ⌊{x}⌋ + ⌊3{y}⌋

This is equivalent to showing:

⌊5{x}⌋ ≥ ⌊3{x}⌋ + ⌊{x}⌋ and ⌊5{y}⌋ ≥ ⌊3{y}⌋ + ⌊{y}⌋

Since both inequalities are symmetric, let's focus on proving:

⌊5{x}⌋ ≥ ⌊3{x}⌋ + ⌊{x}⌋ + ⌊{x}⌋

This inequality holds due to the properties of the floor function. We know 5{x} = 3{x} + 2{x}, so ⌊5{x}⌋ = ⌊3{x} + 2{x}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋. Now, we need to show ⌊2{x}⌋ ≥ ⌊{x}⌋, which is not always true. For example, if {x} = 0.4, then ⌊2{x}⌋ = ⌊0.8⌋ = 0 and ⌊{x}⌋ = 0. However, we made an error in the previous step, it should be ⌊5{x}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋, not ⌊5{x}⌋ ≥ ⌊3{x}⌋ + ⌊{x}⌋ + ⌊{x}⌋. This correction makes our proof much simpler.

By the property ⌊a+b⌋ ≥ ⌊a⌋ + ⌊b⌋, we have ⌊5x}⌋ = ⌊3{x} + 2{x}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋ and similarly ⌊5{y}⌋ = ⌊3{y} + 2{y}⌋ ≥ ⌊3{y}⌋ + ⌊2{y}⌋. Now, we want to show ⌊5{x⌋ + ⌊5y}⌋ ≥ ⌊3{x} + {y}⌋ + ⌊{x} + 3{y}⌋ We already know that ⌊5{x⌋ + ⌊5y}⌋ ≥ ⌊3{x}⌋ + ⌊2{x}⌋ + ⌊3{y}⌋ + ⌊2{y}⌋ We also know the following inequalities ⌊3{x + y}⌋ ≤ ⌊3{x}⌋ + ⌊{y}⌋ ⌊{x} + 3{y}⌋ ≤ ⌊{x}⌋ + ⌊3{y}⌋ Adding them ⌊3{x + y}⌋ + ⌊{x} + 3{y}⌋ ≤ ⌊3{x}⌋ + ⌊{y}⌋ + ⌊{x}⌋ + ⌊3{y}⌋ So, it is enough to show ⌊3{x⌋ + ⌊2x}⌋ + ⌊3{y}⌋ + ⌊2{y}⌋ ≥ ⌊3{x}⌋ + ⌊{y}⌋ + ⌊{x}⌋ + ⌊3{y}⌋ Which simplifies to ⌊2{x⌋ ≥ ⌊{x}⌋ and ⌊2{y}⌋ ≥ ⌊{y}⌋ These are obviously true, because if {x} < 0.5, then both sides are 0, and if {x} >= 0.5, then ⌊2{x}⌋ >= 1, while ⌊{x}⌋ is still 0. Therefore, we have proven the fractional part inequality, and thus, the original inequality holds.
