Интеграция Яндекс Карты В Модальное Окно В Next.js Приложении

Jul 17, 2025 by ADMIN 62 views

Введение

Всем привет, ребята! Сегодня мы с вами разберем, как интегрировать Яндекс Карту в модальное окно в вашем Next.js приложении. Это может показаться сложной задачей на первый взгляд, но не переживайте, мы разложим все по полочкам, и вы увидите, что это вполне реально и даже интересно! В современном веб-разработке интерактивные карты стали неотъемлемой частью многих приложений. Они помогают пользователям находить местоположения, строить маршруты и получать визуальное представление о местности. Яндекс Карты API предоставляет мощные инструменты для работы с картами, и интеграция их в Next.js приложение может значительно улучшить пользовательский опыт. Мы с вами рассмотрим, как правильно подключить API, инициализировать карту, добавить маркеры и, самое главное, как разместить все это в модальном окне, используя, например, Ant Design (Antd). Этот подход позволит вам создавать более гибкие и удобные интерфейсы, где карта появляется только тогда, когда это необходимо, не загромождая основной контент страницы. Так что, если вы хотите добавить интерактивности и удобства в ваше приложение, оставайтесь с нами, и мы начнем!

Подготовка к Интеграции Яндекс Карт в Next.js

Итак, прежде чем мы начнем интегрировать Яндекс Карту в наше Next.js приложение, давайте убедимся, что у нас все готово. Первым делом, нам понадобится создать новый Next.js проект или использовать уже существующий. Если вы только начинаете, вот краткая инструкция, как создать новый проект:

npx create-next-app my-next-app
cd my-next-app
npm install

Эта команда создаст новый проект с базовой структурой Next.js. Далее, нам нужно установить необходимые зависимости. В нашем случае, мы будем использовать Ant Design (Antd) для создания модального окна, а также нам понадобится библиотека для работы с Яндекс Картами API. Вы можете установить Antd следующим образом:

npm install antd

После установки Antd, нам нужно получить ключ API для Яндекс Карт. Это необходимо для использования сервисов Яндекс Карт в вашем приложении. Чтобы получить ключ, вам нужно зарегистрироваться на сайте Яндекс.Карт API и создать новый API-ключ. Этот процесс довольно прост и займет всего несколько минут. Полученный ключ нужно будет добавить в ваш проект, чтобы карта корректно отображалась. Важно помнить, что ключ API является конфиденциальной информацией, и его не следует коммитить в публичные репозитории. Лучше всего хранить его в переменных окружения и получать доступ к нему через process.env. Теперь, когда у нас есть все необходимое, мы можем перейти к следующему шагу – загрузке и инициализации Яндекс Карт.

Загрузка и Инициализация Яндекс Карт

Теперь, когда у нас есть ключ API, давайте разберемся, как загрузить и инициализировать Яндекс Карту в нашем Next.js приложении. Загрузка API Яндекс Карт – это важный шаг, поскольку без него карта просто не отобразится. Мы будем использовать компонент <Script> от Next.js для асинхронной загрузки скрипта Яндекс Карт. Это позволит нам избежать блокировки основного потока и улучшить производительность приложения.

Для начала, создадим компонент, в котором будет отображаться наша карта. Назовем его, например, MapModal. Внутри этого компонента мы будем использовать <Script>, чтобы загрузить API Яндекс Карт. Вот пример кода:

import Script from 'next/script';
import { useState, useEffect, useRef } from 'react';

const MapModal = ({ apiKey }) => {
  const [isLoaded, setIsLoaded] = useState(false);
  const mapRef = useRef(null);

  useEffect(() => {
    if (isLoaded) {
      ymaps.ready(() => {
        const map = new ymaps.Map(mapRef.current, {
          center: [55.75, 37.62],
          zoom: 10,
        });
      });
    }
  }, [isLoaded]);

  const handleLoad = () => {
    setIsLoaded(true);
  };

  return (
    <div>
      <div ref={mapRef} style={{ width: '100%', height: '400px' }} />
      <Script
        src={`https://api-maps.yandex.ru/2.1/?apikey=${apiKey}&lang=ru_RU`}
        strategy="beforeInteractive"
        onLoad={handleLoad}
      />
    </div>
  );
};

export default MapModal;

В этом коде мы используем useState для отслеживания состояния загрузки API и useRef для получения доступа к DOM-элементу, в котором будет отображаться карта. Когда скрипт загружен, мы инициализируем карту с помощью ymaps.ready. Обратите внимание на strategy="beforeInteractive" в компоненте <Script>. Это говорит Next.js загрузить скрипт до того, как страница станет интерактивной, что может улучшить производительность. Теперь, когда мы загрузили и инициализировали карту, давайте перейдем к размещению ее в модальном окне с использованием Antd.

Размещение Яндекс Карты в Модальном Окне с Antd

Отлично, теперь у нас есть компонент с Яндекс Картой, который мы можем отобразить. Следующий шаг – разместить эту карту в модальном окне, используя Antd. Модальные окна – это отличный способ отображения дополнительной информации или интерактивных элементов, таких как карты, без перегрузки основного интерфейса. Antd предоставляет удобный компонент Modal, который мы и будем использовать.

Сначала, импортируем необходимые компоненты из Antd в наш родительский компонент, например, в pages/index.js:

import { Modal, Button } from 'antd';
import { useState } from 'react';
import MapModal from '../components/MapModal';

const HomePage = () => {
  const [isModalVisible, setIsModalVisible] = useState(false);

  const showModal = () => {
    setIsModalVisible(true);
  };

  const handleCancel = () => {
    setIsModalVisible(false);
  };

  return (
    <div>
      <Button type="primary" onClick={showModal}>
        Открыть карту
      </Button>
      <Modal
        title="Яндекс Карта"
        visible={isModalVisible}
        onCancel={handleCancel}
        footer={null}
        width={'80%'}
      >
        <MapModal apiKey="YOUR_API_KEY" />
      </Modal>
    </div>
  );
};

export default HomePage;

В этом коде мы используем useState для управления видимостью модального окна. Кнопка
