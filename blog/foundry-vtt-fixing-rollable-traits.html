Possible Workarounds (While We Wait for a Fix)

For example, if a trait should use the Cool skill, you would roll the trait's base value and then manually add the character's Cool bonus to the result.

  2. Create a Macro: Macros in Foundry VTT are like mini-programs that can automate tasks. You could create a macro that rolls the trait and manually adds the skill bonus.   3. Use a Generic Roll and Add Modifiers: Another approach is to roll a generic dice pool (like a d100) and then manually add the trait's value and the skill bonus as modifiers. This is similar to the first workaround, but it treats the trait more like a bonus to a generic roll rather than the base for the roll itself. It might not fit the flavor of every trait, but it's a functional workaround.

  4. Simplify the Trait: If possible, you could temporarily simplify the trait's mechanics. Instead of linking it to a skill, you could assign it a fixed bonus or value.
