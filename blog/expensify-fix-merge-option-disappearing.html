Initially, the merge option is there, taunting you with its presence. But once that receipt is attached, poof! It vanishes, leaving you scratching your head. The internal Applause team flagged this, with Slottwo12@gmail.com as the affected tester. They were using a Redmi Note 10s running Android 13 when they stumbled upon this vanishing act. The app component involved here is specifically related to money requests, which means it's something that touches a core part of Expensify's functionality. When a key feature like merging expenses becomes unreliable, it can really throw a wrench in the gears of expense management. Workaround and Platform Impact

Okay, so we've got this pesky bug where the merge option disappears, and it's a real pain. But is there a way to sidestep this issue while the Expensify team works on a permanent fix? Unfortunately, as of now, there's no known workaround for this disappearing merge option. It's like being stuck in a maze with only one way out, and that way is a full page refresh. This means that if you find yourself in this situation, the only solution is to refresh the page and navigate back to the expense details. It's not ideal, but it's the only trick we've got in our bag for now. This lack of a quick fix really highlights the importance of getting this bug squashed, because refreshing the page every time you need to merge expenses can quickly become tedious and disrupt the user experience.

Now, let's talk about where this issue is popping up. Bugs don't always play fair, and sometimes they only affect certain platforms or browsers. In this case, the disappearing merge option has been confirmed on Android using mWeb Chrome, and on Windows using Chrome. So, if you're using Expensify on these platforms, you're more likely to encounter this issue. Interestingly, other platforms like the Android app, iOS (both the app and mWeb Safari/Chrome), and MacOS (Chrome/Safari/Desktop) haven't reported this issue. This suggests that the bug might be related to specific rendering or JavaScript behaviors on certain browsers and operating systems. Knowing which platforms are affected helps the Expensify team narrow down the potential causes and focus their debugging efforts. So, if you're on Android mWeb Chrome or Windows Chrome, be extra vigilant for the disappearing merge option and remember that page refresh is your current best friend.
