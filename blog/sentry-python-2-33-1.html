Improved Error Grouping

Another area of improvement is in the handling of errors with dynamic data. Many applications use dynamic data, such as user input or external API responses, which can cause errors to vary slightly from one occurrence to the next. This can make it challenging for Sentry to group these errors effectively. The latest release includes enhancements to the error grouping algorithm that allow it to better handle dynamic data.
