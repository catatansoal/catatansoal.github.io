Understanding the Foundation: Rudin's Problem 1.6

Before we jump into part (d), let's quickly recap the setup. This problem builds the foundation for understanding exponentiation with real exponents, starting from a rational exponent base. We're given a fixed real number b greater than 1. The problem progresses through a series of steps, each building on the previous one, ultimately leading to the definition of b^x for any real number x. This is a classic example of how mathematical concepts are built rigorously, step-by-step.

(a) Rational Exponents: The first part establishes the basics for rational exponents. It shows that if m, n, p, q are integers with n and q positive, and m/n = p/q, then (bm)(1/n) = (bp)(1/q). This is crucial because it ensures that our definition of rational exponents is consistent, regardless of the specific representation of the rational number. Think of it like saying 1/2 is the same as 2/4 – we need to make sure the exponentiation behaves the same way regardless of how we write the fraction. This part often involves clever manipulation of exponent rules and the definition of nth roots.

(b) Defining b^r for Rational r: Next, we define b^r for rational numbers r using the result from part (a). We denote b^(m/n) as (bm)(1/n), which we now know is well-defined. This definition extends the idea of integer exponents to rational exponents. For example, b^(3/2) means the square root of b cubed. This step is essential for bridging the gap between integer exponents, which are relatively straightforward, and real exponents, which require a more sophisticated approach. It's like building a bridge from familiar territory to uncharted lands.

(c) Monotonicity of b^r: This part proves that if r and s are rational numbers with r < s, then b^r < b^s. This establishes the monotonicity of the exponential function for rational exponents, meaning that as the exponent increases, the value of the expression also increases (since b > 1). This is a fundamental property of exponential functions and is essential for extending the definition to real exponents. Imagine the graph of an exponential function – it's always climbing upwards, and this part mathematically proves that behavior for rational inputs. The monotonicity property is a key ingredient in showing that the exponential function behaves predictably and consistently.

The Heart of the Matter: Proving Rudin 1.6(d) – The Exponentiation Rule

Now we arrive at the core of your question: part (d). This is where things get really interesting! Part (d) asks us to show that for rational exponents r and s, the following exponentiation rule holds: b^(r+s) = b^r * b^s. This is one of the most fundamental properties of exponents, and it's crucial for working with exponential functions. It essentially says that multiplying exponential expressions with the same base is equivalent to adding their exponents. But how do we prove it rigorously, starting from the definitions we've built so far?

This is more than just memorizing a rule; it's about understanding why the rule works from the ground up, based on the fundamental properties we've established. Breaking Down the Proof Step-by-Step

The key to tackling this problem is to break it down into smaller, manageable steps. We'll leverage the definitions and properties we've already established in parts (a), (b), and (c). Here's a structured approach to proving b^(r+s) = b^r * b^s:

1. Express r and s as fractions: Since r and s are rational numbers, we can express them as fractions. Let r = m/n and s = p/q, where m, n, p, q are integers and n, q are positive. This is a crucial first step because it allows us to work with the definition of exponentiation for rational exponents that we established in part (b). By expressing r and s in this form, we can manipulate them using the familiar rules of fraction arithmetic. It's like converting ingredients to a common unit before combining them in a recipe.

2. Find a common denominator: To add the fractions, we need a common denominator. The sum r + s can be written as (mq + np) / (nq). This step is a simple application of fraction addition, but it's essential for expressing the exponent (r + s) as a single fraction. Finding a common denominator allows us to combine the exponents into a single term, which is necessary for applying the definition of rational exponents.

3. Apply the definition of b^(r+s): Now, using the definition from part (b), we have b^(r+s) = b^((mq + np) / (nq)) = (b^(mq + np))^(1/(nq)). This step directly applies the definition of exponentiation for rational exponents. We're essentially substituting the expression for (r + s) into our definition. This is a key step in connecting the left-hand side of our equation (b^(r+s)) to an expression that we can further manipulate.

4. Use integer exponent rules: Here's where the properties of integer exponents come into play. We can rewrite b^(mq + np) as b^(mq) * b^(np). This is a fundamental property of exponents that holds for integers. It allows us to separate the exponent into two terms, each of which we can deal with more easily. This step is crucial for breaking down the complex exponent into simpler components.

5. Rewrite using fractional exponents: Now we reverse the process from step 3. We can rewrite b^(mq) as (bm)q and b^(np) as (bp)n. This step is essentially undoing the distribution of exponents. By grouping the exponents in this way, we're setting ourselves up to use the definition of fractional exponents again. It’s like reassembling the pieces of a puzzle in a different configuration.

6. Substitute back into the expression: Substitute these expressions back into the equation from step 3. We now have (b^(mq + np))^(1/(nq)) = ((bm)q * (bp)n)^(1/(nq)). This step is a straightforward substitution, but it brings us closer to our goal. We're gradually transforming the left-hand side of the equation into something that resembles the right-hand side.

7. Apply the power of a product rule: The next key step is to use the rule that the nth root of a product is the product of the nth roots. This allows us to write ((bm)q * (bp)n)^(1/(nq)) = ((bm)q)^(1/(nq)) * ((bp)n)^(1/(nq)). This is a crucial step because it separates the two terms inside the parentheses, allowing us to simplify them individually. It's like dividing a complex problem into smaller, more manageable subproblems.

8. Simplify the exponents: Now we simplify the exponents. We have ((bm)q)^(1/(nq)) = (bm)(q/(nq)) = (bm)(1/n) and ((bp)n)^(1/(nq)) = (bp)(n/(nq)) = (bp)(1/q). This step involves simplifying fractions and applying the power of a power rule. We're essentially reducing the exponents to their simplest forms, which makes the expression much easier to work with. It’s like trimming away the excess to reveal the core structure.

9. Use the definition of b^r and b^s: Finally, we recognize that (bm)(1/n) = b^(m/n) = b^r and (bp)(1/q) = b^(p/q) = b^s. This step brings us full circle, connecting the expression back to our original definitions of b^r and b^s. We've successfully transformed the left-hand side of the equation into the right-hand side!

10. Conclude the proof: Putting it all together, we have shown that b^(r+s) = b^r * b^s. We've successfully proven the exponentiation rule for rational exponents! This is the grand finale of our proof, where all the pieces fall into place. We've demonstrated, step-by-step, that the exponentiation rule holds true for rational exponents, based on the fundamental definitions and properties we've established.
